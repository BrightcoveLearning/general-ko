문자열 인코더

3 가지 방법으로 문자열을 URI 인코딩하는 유틸리티.

제안하는 메서드 이름을 추가합니다. 일반적으로 URL에 문자열을 포함해야 할 때 유용합니다. 어떤 것이 적절한 지 상황에 따라 다릅니다. 다음은 유용한 정보를 보여주는 몇 가지 예입니다.

  • escape가장 오래된 방법이며, 가장 적은 방법입니다. 앱에서 사용할 기회가 없었습니다.
  • encodeURI일반적으로 URL 매개 변수의 값으로 전달되는 문자열에 적합합니다.
  • encodeURIComponent가장 철저한 인코더이지만 일반적으로 URL 도메인/경로의 일부로 사용되는 문자열에 대해서만 필요합니다. 또는 URL을 매개 변수로 전달해야하는 경우 (API 요청을 프록시를 통해 자주 전달합니다.아약스

인코딩할 문자열:

인코딩하세요!
