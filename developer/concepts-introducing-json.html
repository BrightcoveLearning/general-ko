JSON으로 시작

JSON (JavaScript Object Notation) 파일 형식은 서버와 웹 애플리케이션간에 구조화 된 데이터를 직렬화하고 전송하는 데 사용되는 텍스트 기반의 개방형 표준 형식입니다. JSON 형식은 사람이 읽고 쓰기가 쉽습니다. 기계가 파싱하고 생성하는 것도 쉽습니다. JavaScript 프로그래밍 언어의 하위 집합을 기반으로하지만 완전히 언어 독립적입니다. JSON 형식은 XML보다 작고 빠르며 파싱하기 쉽습니다. 이러한 속성 때문에 JSON 형식은 이상적인 데이터 교환 언어입니다.

JSON 형식의 데이터 유형은 다음과 같습니다.

  • 번호 -JavaScript의 배정 밀도 부동 소수점
  • 끈 -백 슬래시 이스케이프가있는 큰 따옴표 유니 코드
  • 부울 -참 또는 거짓
  • 정렬 -대괄호로 묶인 쉼표로 구분 된 값의 순서가 지정된 시퀀스
  • Object - 키와 값을 구분하는 콜론 “:”이 있는 키:값 쌍의 정렬되지 않은 컬렉션입니다. 중괄호로 묶인 쉼표로 구분 된 목록입니다.
  • 없는 -null 값

다음 예는 JSON 형식으로 개인의 연락처 정보를 설명합니다.

{
    "firstName": "John",
    "lastName": "Smith",
    "age": 25,
    "address": {
        "streetAddress": "21 2nd Street",
        "city": "New York",
        "state": "NY",
        "postalCode": 10021
    },
    "phoneNumbers": [
        {
            "type": "home",
            "number": "212 555-1234"
        },
        {
            "type": "fax",
            "number": "646 555-4567"
        }
    ]
}

JSON 형식에 대한 자세한 내용은 JSON 소개그리고자바스크립트의 JSON서류.

JSON 메서드 사용

JSON 형식은 JavaScript 언어와 매우 밀접한 관련이 있기 때문에 (U + 2028 및 U + 2029 문자는 두 언어에서 다르게 처리됨) 거의 노력없이 언어로 사용할 수 있습니다. JSON 텍스트 형식은 JavaScript 개체를 만드는 데 사용되는 코드와 동일한 구문을 사용합니다. JSON 개체에는 값을 JSON 형식으로 변환하고 JSON 형식을 값으로 변환하는 메서드가 포함되어 있습니다.

JSON 객체에는 다음을 포함하는 여러 메서드가 있습니다.

  • JSON 문자열을 JavaScript 객체 또는 배열로 변환하는parse()메서드입니다. 
    var jsonstring = '{"firstName":"John","lastName":"Smith","phone":["555-0100","555-0120"]}';
var contact = JSON.parse(jsonstring);
console.log(contact.lastName + ", " + contact.firstName);
//Output: Smith, John

    자세한 내용은 JSON.parse 함수 (자바스크립트)문서를 참조하십시오.

  • JavaScript 값 (일반적으로 객체 또는 배열) 을 JSON 문자열로 변환하는stringify()메서드입니다.

    예 1:

    var contact = new Object();
contact.firstName = "John";
contact.lastName = "Smith";
contact.phone = ["555-0100","555-0120"];
var jsonstring = JSON.stringify(contact);
console.log(jsonstring);
//Output: { "firstName":"John","lastName":"Smith","phone":["555-0100","555-0120"] }

    예 2: (이스케이프 문자는 내부 문자열에 사용됩니다.)

    var foo = new Object();
foo.template = '<p class="bar">Hello</p>';
var jsonstring = JSON.stringify(foo);
console.log(jsonstring);
//Output: {"template":"<p class=\"bar\">Hello</p>"}

    자세한 내용은 JSON.stringify 함수 (자바스크립트)문서를 참조하십시오.

JSON 텍스트 만들기

JSON 형식의 텍스트 파일을 많이 사용하고 만드는 경우 JSON 편집기를 사용하여 파일을보고, 편집하고, 형식을 지정할 수 있습니다.

다운로드 가능한 JSON 편집기는 다음과 같습니다.

  • 파워 JSON 편집기
  • 이클립스 JSON 편집기 플러그인
  • Chrome 웹 스토어 JSON 편집기

온라인 JSON 편집기도 있습니다.

  • JSON 편집기 온라인
  • JSON메이트
  • JSON 파서 온라인

JSON 형식 파일로 작업하는 데 도움이되는 무료 도구가 많이 있습니다. 브라우저를 사용하여 다른 응용 프로그램을 검색하십시오.
