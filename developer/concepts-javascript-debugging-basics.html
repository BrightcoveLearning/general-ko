개념: 자바스크립트 디버깅 기본

이 항목에서는 사용자 지정 JavaScript 코드를 Brightcove Player에 추가 할 때 디버깅하는 데 도움이되는 몇 가지 기본 개념을 배우고 다음 솔루션을 다룹니다.

  • 기본 비디오 플레이어 코드 시작하기
  • 브라우저 콘솔에 메시지 표시

플레이어 코드

코드를 테스트하고 디버깅할 때 페이지 내 임베드 플레이어 구현 (Studio의고급코드라고 함) 을 사용하는 것이 가장 유용합니다. 이것은 플레이어와 상호 작용하는 가장 명확한 방법을 제공합니다.

JavaScript를 사용하는 것은 거의 확실하게 테스트 및 디버깅의 일부이므로 JavaScript를 사용하도록 코드를 설정하는 것이 좋습니다. 이 작업은ready이벤트를 사용하여 표준 방식으로 수행됩니다. 인페이지 포함 구현 코드와ready이벤트가 모두 여기에 표시됩니다.

  • 11-18번 라인: 표준 인 페이지 소스 플레이어 코드
  • 20-24번 라인: JavaScript를 사용할 표준 스크립트 블록
  • 11번, 21번 라인: 플레이어가id추가 및 사용되었습니다. 이는 표준 인 페이지 소스 구현 코드에 추가 된 것입니다.
<!DOCTYPE html>
<html>

<head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
    <title></title>
</head>

<body>
    <h1>Sample Player</h1>
    <video-js id="myPlayerID"
    style="width: 640px; height: 360px;"
    data-video-id="4845798282001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js" controls></video-js>
    <script src="//players.brightcove.net/1752604059001/default_default/index.min.js"></script><script>
        videojs.getPlayer('myPlayerID').ready(function() {
            var myPlayer = this;
            console.log('EVENT: Player ready');
            myPlayer.on('loadedmetadata', function(evt) {
                console.log('EVENT: Metadata loaded');
            });
        });
    </script>
</body>

</html>

메시지 표시

코드를 테스트하고 디버깅 할 때 텍스트 문자열과 개체 값을 인쇄하는 것이 유용합니다. console객체를 사용하여 이 작업을 수행할 수 있습니다. 이 개체는 JavaScript 언어의 일부가 아니지만 런타임 환경의 일부이며 대부분의 최신 브라우저에서 사용할 수 있습니다. Firefox에서는 Firebug 확장과 함께 제공됩니다. WebKit 브라우저 (Safari 및 Chrome)에서는 Web Inspector / DevTools의 일부입니다. Internet Explorer에서는 IE 11부터 개발자 도구의 일부입니다.

console객체에는 다음과 같은 여러 메서드가 있습니다.

  • 로깅 정보의 일반적인 출력log()방법입니다. 이것은 문자열 대체 및 추가 인수를 허용합니다. 
    console.log("test",1,{},[1,2,3]);
  • 객체 속성의 대화식 목록을 표시하는dir()메서드입니다. 
    console.dir({one:1, two: {three:3}});

다음은 메시지 를 표시하는 콘솔의 예제 코드와 스크린샷입니다. console.log()

다음은 각 브라우저에서 콘솔 개체를 사용하는 방법에 대한 세부 정보입니다.

크롬

Chrome 브라우저에서 메시지보기

Chrome 브라우저에서 샘플 코드를 실행합니다. 메시지를 보려면 DevTools 패널을 엽니 다.

  1. Chrome 브라우저에서 Chrome 맞춤설정 및 제어 상.
  2. 도구 > 자바스크립트 콘솔을 선택합니다 .
  3. DevTools 패널에서콘솔메뉴 항목을 선택한 상태에서 콘솔 메시지가 여기에 표시됩니다.

Chrome 개발자도구 사용에 대한 세부 정보를 확인하세요 .

원정 여행

Safari 브라우저에서 메시지보기

Safari 브라우저에서 샘플 코드를 실행하십시오. 메시지를 보려면 개발자 도구 패널을 엽니 다.

  1. Safari 메뉴 막대에현상메뉴 항목이 표시되지 않으면 다음 단계를 수행하십시오.
    1. Safari > 환경 설정 > 고급을 선택합니다
    2. 메뉴 막대에 현상 메뉴표시옵션을 선택합니다.
  2. Safari 메뉴 막대에서 개발하다 > 오류 콘솔 표시.
  3. 오류 콘솔 패널에 콘솔 메시지가 표시되어야합니다.

Safari 웹 인스펙터 사용에 대한 세부 정보를확인합니다.

Firefox

Firefox 브라우저에서 메시지보기

Firefox 브라우저에서 샘플 코드를 실행하십시오. 메시지를 보려면 웹 콘솔 패널을여십시오.

  1. Firefox 메뉴 모음에서도구 > 웹 개발자 > 웹 콘솔을 선택합니다 .
  2. 웹 콘솔 패널에 콘솔 메시지가 표시되어야합니다.
