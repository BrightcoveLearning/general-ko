소개

Wireshark는 예를 들어 브라우저 개발자 도구의 네트워크 부분을 통해 얻을 수있는 것보다 훨씬 더 깊이있는 네트워크 프로토콜을 분석하기위한 인기있는 오픈 소스 애플리케이션입니다. 이는 Brightcove Player와 같은 인터넷 애플리케이션 또는 API를 통해 Brightcove와 상호 작용하는 앱의 문제를 진단하는 데 매우 유용 할 수 있습니다.

네트워크에 대한 자신의 지식이 Wireshark에서 얻은 정보를 완전히 이해하기에 충분하지 않더라도 정보 캡처는 Brightcove 지원이 문제를 진단하는 데 큰 도움이 될 수 있으며 지원 담당자는 캡처가 가능한지 물어볼 수 있습니다. Wireshark의 네트워크 데이터를 통해 더 나은 도움을받을 수 있습니다.

아래 단계는 Wireshark의 기본 설치 및 사용을 안내합니다.

설치

  1. Wireshark가 아직 설치되어 있지 않은 경우 https://www.wireshark.org/#download 로 이동하여플랫폼과 일치하는 설치 프로그램을 다운로드하십시오.
  2. 설치 프로그램을 실행하십시오. 옵션이 표시되면 사용 가능한 모든 구성 요소를 설치합니다. (옵션은 릴리스 및 플랫폼에 따라 다릅니다.)

Wireshark 사용

중요:시작하기 전에 인터넷에 액세스하는 애플리케이션을 최대한 많이 종료하는 것이 가장 좋습니다 (예: 이메일 클라이언트, IM 또는 소셜 미디어 앱, Dropbox, Evernote 및 기타 클라우드 기반 스토리지 앱). 이렇게하면 진단하려는 문제와 관련이없는 네트워크 전송의 "노이즈"를 최소화하는 데 도움이됩니다.

  1. Wireshark를 시작합니다.
  2. 캡처 필터를 선택하십시오. 원하는 필터가 트래픽의 대부분 또는 최소한 좋은 부분을 보여줄 것입니다.
  3. 데이터 캡처를 시작할 필터를 두 번 클릭합니다.
  4. 데이터 캡처를 실행 한 상태에서 플레이어를로드하거나 앱을 실행하는 등 진단하려는 문제를 재현합니다.
  5. 문제 재현이 완료되면 Wireshark에서 데이터 캡처 중지 버튼을 클릭합니다.
  6. 이제 캡처 파일 저장 버튼을 클릭하여 Wireshark에서 데이터를 내 보냅니다.
  7. 컴퓨터에 기본 형식으로 파일을 저장합니다.
  8. 발생한 문제를 자세히 설명하는 지원 요청으로 파일을 Brightcove 지원에 전달하십시오.
