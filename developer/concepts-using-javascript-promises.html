개념: 자바 스크립트 약속 사용

이 항목에서는 일부 Brightcove Player 메서드에서 반환되는 JavaScript Promise를 사용하는 방법을 알아 봅니다.

간단한 약속

이 문서의 목적은 아니지만Promise JavaScript를 작성하는 방법을 가르쳐주는 것은 아니지만 어떻게 사용하는지 이해하려면 얼마나 간단한 지 확인하는 것이 유익합니다. Promise코드화되어 있습니다.

다음에서는 작업이 완료되었는지 여부를 나타내는 문자열을 반환하는 가 만들어집니다. Promise 핵심은Promise.resolve() or Promise.reject()메소드가 호출된다는 것입니다. 이 경우 두 메서드 모두 간단한 문자열이 전달되지만 이것은 객체가 될 수 있습니다.

참고: 이 간단한 예에서는 설명되지 않았지만Promise개체를 나타냅니다최종 완료 (또는 실패) 비동기 작업 , 그리고 그 결과 값. 이런 식으로 a Promise는 개념상 콜백 함수와 비슷합니다.
          let promiseToFinishTask = new Promise(function(resolve, reject) {
          
            //Business logic to set taskComplete boolean
            let taskComplete = true;
          
            if (taskComplete) {
              resolve('Yea, I am done');
            } else {
              reject('Not quite done');
            }
          });

이제 일부 브라이트코브 플레이어 API 메서드 호출을 통해 반환된Promise정보를 처리하는 방법을 확인할 수 있습니다. 기본 처리는 다음과 같습니다.

다음과 같습니다. 
          promiseName.then(
            // Function call for promise resolved
          ).catch(
            // Function call for promise rejected
          )

위의 간단한 예제를 처리하면 다음과 같이 나타납니다.

          promiseToFinishTask.then(function(fromResolve) {
            console.log(fromResolve);
          }).catch(function(fromReject) {
            console.log(fromReject);
          });

Promise가 확인되면then코드가 실행되고Promisecatch코드가 실행되면 거부됩니다. 예를 들어 문자열 중 하나는그래, 난 끝났어또는완료되지 않음설정하는 비즈니스 로직을 기반으로 나타납니다. taskComplete변수의 값.

Brightcove 예

브라이트코브 플레이어play()메서드는 플레이어가 현재 플레이어에 있는 비디오를 자동 재생할 수 있는지 확인하는 데 사용할 수 있는 Promise를 반환할 수 있습니다. 를 다루는 코드는 다음과 같이 강조Promise표시됩니다.

          var myPlayer = videojs.getPlayer('myPlayerID');
          myPlayer.on('loadedmetadata',function() {
            var promise = myPlayer.play();
            if (promise !== undefined) {
              promise.then(function() {
                // Autoplay started!
              }).catch(function(error) {
                // Autoplay was prevented.
              });
            }
          });

이를 기반으로 비디오가 자동 재생되는지 여부에 따라 동작을 코딩 할 수 있습니다. 가능한 경우 오디오와 함께 비디오를 재생하는 특정 사용 사례는특정 사용 사례섹션자동 재생 고려 사항문서.

자바스크립트 프로미스에 대한 자세한 내용은 MDN 프로미스문서를 참조하십시오.

Promise콜백 대

이 섹션에서는 Catalog의get()메서드를 사용하여 a Promise와 콜백 함수 사용 간의 구문 차이를 보여 줍니다. 이get()방법은 두 가지 방법 중 하나에서 사용할 수 있습니다.

약속 구문

다음은get()메서드 호출을 보여주는 코드 스니펫으로 응답을 a로 처리합니다Promise .

          myPlayer.catalog.get(catalogParams).then(function(videosReturned){
            console.log('videosReturned',videosReturned);
            myPlayer.playlist(videosReturned);
          }).catch(function(errorObj){
            console.log('errorObj',errorObj);
          })

콜백 함수 구문

다음은get()메서드 호출을 보여주는 코드 조각이며 콜백 함수로 응답을 처리합니다.

            myPlayer.catalog.get(catalogParams,function(errorObj,videosReturned){
            console.log('errorObj',errorObj);
            console.log('videosReturned',videosReturned);
            myPlayer.playlist(videosReturned);
          });
