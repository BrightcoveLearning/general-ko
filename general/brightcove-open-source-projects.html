페이지 내용

    브라이트코브 오픈소스 프로젝트

    아래는 현재 브라이트코브 기술을 위한 모든 오픈 소스 프로젝트를 범주별로 그룹화하여 나열한 것입니다.
    오픈 소스 프로젝트는 브라이트코브에서 공식적으로 지원하지 않습니다.

    브라이트코브 오픈 소스에 오신 것을 환영합니다. 브라이트코브 직원이나 브라이트코브 커뮤니티 구성원이 제작한 여러 프로젝트를 통해 브라이트코브 개발 속도를 높일 수 있습니다.

    프로젝트에 기여하는 데 관심이 있다면 Github의 개별 프로젝트 사이트로 이동하십시오. 여기에 나열하고 싶은 프로젝트가 있다면저희에게 연락하십시오 .

    현재 API

    1. API 요청 문제 해결 스크립트

      브라이트코브 API에 대한 HTTP 요청 및 응답 문제를 해결합니다.

      https://github.com/BrightcoveOS/api-request-troubleshooting-script

    2. 브라이트코브 업로더

      브라이트코브 인제스트를 위한 비밀 소스와 함께 s3에 직접 멀티파트 업로드를 위한 프론트엔드 모듈입니다.

      https://github.com/BrightcoveOS/evaporate-brightcove

    3. 브라이트코브 재생 API 래퍼

      이것은 테레사 뉴먼이 작성한브라이트코브 재생 API용 PHP 래퍼입니다.

      https://github.com/BrightcoveOS/Brightcove-Playback-API-Wrapper

    4. 브라이트코브 동적 인제스트용 PHP 래퍼

      이 프로젝트는 브라이트코브를 애플리케이션에Dynamic Ingest API통합하기 위한 출발점을 제공합니다. 풀 기반 인제스트 또는 소스 파일 업로드를 사용하여 비디오 및 관련 미디어를 간편하게 추가하고 업데이트할 수 있습니다.

      https://github.com/BrightcoveOS/php-wrapper-dynamic-ingest

    5. API 외관

      이 겸손한 라이브러리는 브라이트코브의 서버 API에 대한 단순한 외관을 목표로 합니다. 폭도에게 추가 기능이 필요하기 때문에 여기에 추가됩니다.

      https://github.com/nwbb/node-brightcove

    6. 플레이어 관리 API 클라이언트

      이 모듈은 브라이트코브 플레이어 관리 API에 액세스하는 데 사용할 수 있습니다. 기본 API에 대한 문서는여기에서찾을 수 있습니다.

      https://github.com/BrightcoveLearning/Player-Management-Client

    7. 브라이트코브 API 예제

      이 프로젝트는 새로운 브라이트코브 API를 C #과 함께 사용하는 방법을 보여줍니다.

      이것은 데스크톱 환경 (테스트 됨) 에서 작동하지만 CORS로 인해 ASP.NET 페이지에서 사용하면 작동하지 않을 수 있습니다. 이 문제가 발생하면이 문제를 해결할 수있는 유일한 방법은 모든CMS API예제 페이지에서 브라이트코브가 제공하는 PHP 프록시를 사용하는 것입니다.

      모든 코드는 그대로 그리고 순전히 예제로 제공됩니다. 자신의 책임하에 사용하십시오. 어떠한 방식, 형태 또는 모양으로도 지원이 제공되지 않습니다.

      https://github.com/mikelau13/BrightcoveAPI

    8. 플레이어 관리 API 클라이언트

      이 모듈은 브라이트코브 플레이어 관리 API에 액세스하는 데 사용할 수 있습니다. 기본 API에 대한 문서는여기에서찾을 수 있습니다.

      https://github.com/BrightcoveLearning/Player-Management-Client

    플레이어 및 SDK

    1. Video.js 플러그인 프레임 워크 생성기

      이것은 video.js 플러그인에 대한 유만 생성기입니다. video.js 코어 팀의 권장 사항과 브라이트코브에서 플러그인을 개발하기 위한 도구 및 규칙을 기반으로 합니다.

      https://github.com/videojs/generator-videojs-plugin

    시스템 통합

    1. 브라이트코브 비디오 커넥트

      SharePoint 2013용 브라이트코브 비디오 연결을 통해 게시자는 SharePoint에서 직접 비디오 및 비디오 재생 목록을 추가, 찾아보기 및 관리할 수 있습니다. 플레이어 구성 요소를 사용하면 브라이트코브 비디오 클라우드 자산을 SharePoint 페이지에 직접 쉽게 통합할 수 있습니다.

      http://go.brightcove.com/bc-video-connect-sharepoint2013

    2. 드루팔 브라이트코브 비디오 커넥트

      브라이트코브 Video Cloud는 모든 디바이스에서 디지털 고객에게 전문적인 품질의 비디오를 제공하기 위한 비디오 호스팅 및 퍼블리싱 솔루션입니다. 사용자 지정 가능한 플레이어, HD 스트리밍, 타사 CDN과 동기화, YouTube와 같은 기능을 제공합니다. 드루팔용 브라이트코브 비디오 커넥트를 사용하면 Drupal이 제공하는 웹 사이트에서 비디오 콘텐츠를 효율적으로 추가하고 관리할 수 있습니다.

      https://www.drupal.org/project/brightcove

    3. 브라이트코브 비디오 워드 프레스에 연결

      WordPress 내에서 브라이트코브 비디오 및 플레이리스트 라이브러리를 기본적으로 처리하고자 하십니까? 10up.com에 의해 개발된이 플러그인으로, 당신은 여러 계정 및 비디오 라이브러리를 처리 비디오를 업로드하고 재생 목록에 추가 워드 프레스 관리 인터페이스 내에서 비디오와 모든 단축을 렌더링 할 수있는 능력을 가지고 있습니다.

      https://wordpress.org/plugins/brightcove-video-connect/

    4. 셰어포인트 2013 커넥터

      SharePoint 2013과 여러 비디오 클라우드 계정 간의 통합을 제공합니다.

      https://github.com/brightcove/sharepoint-connector

    5. Adobe 경험 관리자를 위한 브라이트코브 비디오 커넥트

      AEM (Adobe Experience Manager) 용 브라이트코브 비디오 커넥트는 브라이트코브 비디오 클라우드와 AEM 간의 워크플로를 통합하는 가장 쉬운 방법입니다. AEM 사용자는 브라이트코브 비디오 연결을 사용하여 AEM 사용자 인터페이스 내에서 직접 비디오를 추가, 찾아보기 및 관리할 수 있습니다.

      http://go.brightcove.com/bc-video-connect-aem

    6. CQ5 구성 요소

      CQ5 구성 요소는 브라이트코브를 통합합니다.

      기술: CQ5

      https://www.github.com/coresecure/Adobe-CQ5-Brightcove-Connector

    7. 세일즈포크 리프트

      Salesforce는 데이터 저장소에서 영업 사원으로 대규모 데이터 동기화를 용이하게하기 위해 대량 API를 제공합니다. 이 프로젝트는 루비 응용 프로그램에서 세일즈 포스 대량 API를 사용하는 보석을 제공합니다.

      기술: 루비 세일즈포스

      https://www.github.com/brightcove/salesforklift

    여러 가지 잡다한

    이 프로젝트는 다른 범주에 맞지 않습니다.

    1. 펄 커먼즈

      이 프로젝트는 다른 Perl 프로젝트에서 사용할 수있는 공통 클래스와 유틸리티를 제공합니다. 여기에는 타사 라이브러리와 브라이트코브 오픈 소스 라이브러리가 포함될 수 있습니다.

      기술: 펄

      https://www.github.com/BrightcoveOS/Perl-Commons

    2. 그래파이트 피클 메트릭

      코다 헤일로 확장

      기술: 흑연, 자바

      https://www.github.com/BrightcoveOS/metrics-graphite-pickle

    3. LG 샘플 앱

      브라이트코브 계정의 여러 재생 목록을 표시하는 LG 비디오 애플리케이션입니다. 응용 프로그램은 최소한의 구성이 필요하도록 설계되었으며 고객은 원하는 기능에 따라 UI를 변경하거나 기능을 추가/제거하기 위해 사용자 지정을 할 수 있습니다.

      기술: LG, 텔레비젼

      https://www.github.com/BrightcoveOS/LG-Sample-App

    4. 로쿠 샘플 앱

      이 템플릿은 Roku에 브라이트코브 콘텐츠를 게시하는 데 있어 바로 시작할 수 있습니다. 이 응용 프로그램을 사용하여 게시를 시작하려면 다음이 필요합니다.

      기술: 로쿠 테레비TV

      https://www.github.com/BrightcoveOS/Roku-Sample-App

    5. 자바 커먼즈

      이 프로젝트는 다른 Java 프로젝트에서 사용할 수있는 공통 클래스와 유틸리티를 제공합니다. 여기에는 타사 라이브러리와 브라이트코브 오픈 소스 라이브러리가 포함될 수 있습니다.

      기술: Java

      https://www.github.com/BrightcoveOS/Java-Commons

    6. 다이아몬드

      Diamond는 시스템 메트릭을 수집하여 그래파이트에 게시하는 파이썬 데몬입니다. CPU, 메모리, 네트워크, I/O, 로드 및 디스크 메트릭을 수집 할 수 있습니다. 또한 거의 모든 소스에서 메트릭을 수집하기 위한 사용자 지정 수집기를 구현하기 위한 API를 제공합니다.

      기술: 흑연, 파이썬

      https://github.com/python-diamond/Diamond
