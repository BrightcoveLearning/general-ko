브라이트코브 오픈 소스 프로젝트

다음은 카테고리별로 분류 된 현재 Brightcove 기술에 대한 모든 오픈 소스 프로젝트 목록입니다.
오픈 소스 프로젝트는 Brightcove에서 공식적으로 지원하지 않습니다.

Brightcove 오픈 소스에 오신 것을 환영합니다. Brightcove 직원 또는 Brightcove 커뮤니티 구성원이 Brightcove 개발 속도를 높이는 데 도움이되는 여러 프로젝트를 여기에서 찾을 수 있습니다.

프로젝트에 기여하고 싶다면 Github의 개별 프로젝트 사이트로 이동하세요. 여기에 나열하려는 프로젝트가 있는 경우 당사에문의하십시오 .

현재 API

  1. API 요청 문제 해결 스크립트

    Brightcove API와의 HTTP 요청 및 응답 문제를 해결합니다.

    https://github.com/BrightcoveOS/api-request-troubleshooting-script

  2. Brightcove 업 로더

    Brightcove 수집을위한 비밀 소스를 사용하여 s3에 직접 멀티 파트 업로드를위한 프런트 엔드 모듈.

    https://github.com/BrightcoveOS/evaporate-brightcove

  3. Brightcove Playback API 래퍼

    이것은 PHP 래퍼입니다. 브라이트코브 재생 API테레사 뉴먼이 썼다.

    https://github.com/BrightcoveOS/Brightcove-Playback-API-Wrapper

  4. Brightcove Dynamic Ingest 용 PHP 래퍼

    이 프로젝트는 브라이트코브다이내믹 인제스트 API를 애플리케이션에통합하기 위한 출발점을 제공합니다. 풀 기반 수집 또는 소스 파일 업로드를 사용하여 비디오 및 관련 미디어를 추가하고 업데이트하는 간단한 방법을 제공합니다.

    https://github.com/BrightcoveOS/php-wrapper-dynamic-ingest

  5. API 파사드

    이 겸손한 라이브러리는 Brightcove의 서버 API에 대한 단순한 외관을 목표로합니다. 몹이 추가 기능을 요구함에 따라 여기에 추가됩니다.

    https://github.com/nwbb/node-brightcove

  6. 플레이어 관리 API 클라이언트

    이 모듈은 Brightcove Player Management API에 액세스하는 데 사용할 수 있습니다. 기본 API에 대한 문서는여기에서 찾을 수있습니다.

    https://github.com/BrightcoveLearning/Player-Management-Client

  7. CSharp를 사용하는 Brightcove API 예제

    이 프로젝트는 새로운 Brightcove API를 C #과 함께 사용하는 방법을 보여줍니다.

    이것은 데스크톱 환경 (테스트 됨)에서 작동하지만 CORS로 인해 ASP.NET 페이지에서 사용되는 경우 작동하지 않을 수 있습니다. 이 문제를 해결할 수 있는 유일한 방법은 브라이트코브가 제공하는 PHP 프록시를 모든 CMS API예제 페이지에서 사용하는 것입니다.

    모든 코드는있는 그대로 제공되며 순전히 예제로 제공됩니다. 자신의 책임하에 사용하십시오. 어떤 방식, 형태 또는 모양으로도 지원이 제공되지 않습니다.

    https://github.com/houseofmackee/BrightcoveCS

  8. Brightcove API Python 도구

    이 저장소에는 Brightcove 관련 API를 처리하고 전체 비디오 라이브러리를 반복하고 비디오에 대한 작업을 수행하는 등의 작업을 단순화하기 위해 Python으로 작성된 도구와 모듈이 포함되어 있습니다.

    테스트된 파이썬 버전: 3.8.3

    모든 코드는있는 그대로 제공되며 순전히 예제로 제공됩니다. 자신의 책임하에 사용하십시오. 어떤 방식, 형태 또는 모양으로도 지원이 제공되지 않습니다.

    https://github.com/houseofmackee/BrightcovePY

  9. 플레이어 관리 API 클라이언트

    이 모듈은 Brightcove Player Management API에 액세스하는 데 사용할 수 있습니다. 기본 API에 대한 문서는여기에서 찾을 수있습니다.

    https://github.com/BrightcoveLearning/Player-Management-Client

플레이어 및 SDK

  1. Video.js 플러그인 프레임 워크 생성기

    이것은 video.js 플러그인을위한 독자적인 Yeoman 생성기입니다. 이는 video.js 핵심 팀의 권장 사항과 Brightcove의 플러그인 개발을위한 도구 및 규칙을 기반으로합니다.

    https://github.com/videojs/generator-videojs-plugin

시스템 통합

  1. SharePoint 2013 용 Brightcove Video Connect

    SharePoint 2013 용 Brightcove Video Connect를 사용하면 게시자가 SharePoint에서 직접 비디오 및 비디오 재생 목록을 추가, 탐색 및 관리 할 수 있습니다. 플레이어 구성 요소를 사용하면 Brightcove Video Cloud 자산을 SharePoint 페이지에 직접 쉽게 통합 할 수 있습니다.

    http://go.brightcove.com/bc-video-connect-sharepoint2013

  2. Drupal 용 Brightcove 비디오 연결

    Brightcove Video Cloud는 모든 장치의 디지털 청중에게 전문적인 품질의 비디오를 제공하기위한 비디오 호스팅 및 게시 솔루션입니다. 고도로 사용자 정의 가능한 플레이어, HD 스트리밍, 타사 CDN 및 YouTube와의 동기화를 제공합니다. Brightcove Video Connect for Drupal을 사용하면 Drupal 기반 웹 사이트에서 비디오 콘텐츠를 효율적으로 추가하고 관리 할 수 있습니다.

    https://www.drupal.org/project/brightcove

  3. WordPress 용 Brightcove Video Connect

    WordPress 내에서 기본적으로 Brightcove 비디오 및 재생 목록 라이브러리를 처리하려고합니까? 10up.com에서 개발 한이 플러그인을 사용하면 여러 계정 및 비디오 라이브러리를 처리하고, 비디오를 업로드하고, 재생 목록에 추가하고, 비디오와 함께 단축 코드를 렌더링하는 등 모든 작업을 WordPress 관리 인터페이스 내에서 수행 할 수 있습니다.

    https://wordpress.org/plugins/brightcove-video-connect/

  4. SharePoint 2013 커넥터

    SharePoint 2013과 여러 Video Cloud 계정 간의 통합을 제공합니다.

    https://github.com/brightcove/sharepoint-connector

  5. Adobe Experience Manager 용 Brightcove Video Connect

    AEM (Adobe Experience Manager) 용 Brightcove Video Connect는 Brightcove Video Cloud와 AEM 간의 워크 플로우를 통합하는 가장 쉬운 방법입니다. Brightcove Video Connect를 사용하여 AEM 사용자는 AEM 사용자 인터페이스 내에서 직접 비디오를 추가, 탐색 및 관리 할 수 있습니다.

    http://go.brightcove.com/bc-video-connect-aem

  6. CQ5 구성 요소

    CQ5 구성 요소는 Brightcove를 통합합니다.

    기술: CQ5

    https://www.github.com/coresecure/Adobe-CQ5-Brightcove-Connector

  7. 판매 지게차

    Salesforce는 대량 API를 제공하여 데이터 저장소에서 Salesforce로 대규모 데이터 동기화를 용이하게합니다. 이 프로젝트는 Ruby 앱에서 Salesforce Bulk API를 사용하기위한 gem을 제공합니다.

    기술: Salesforce, Ruby

    https://www.github.com/brightcove/salesforklift

해석학

  1. Adobe Analytics

    Adobe Analytics 플러그인은 플레이어 모듈을 사용하여 쉽게 구성 할 수 있습니다. 자세한 내용은 Adobe Analytics 이벤트 추적구성을 참조하십시오 .

  2. Google Analytics

    Google Analytics 플러그인은 플레이어 모듈을 사용하여 쉽게 구성 할 수 있습니다. 자세한 내용은 Google 애널리틱스 이벤트 추적구성을 참조하십시오 .

여러 가지 잡다한

이 프로젝트는 다른 카테고리에 맞지 않습니다.

  1. Perl Commons

    이 프로젝트는 다른 Perl 프로젝트에서 사용할 수있는 공통 클래스와 유틸리티를 제공합니다. 여기에는 타사 라이브러리와 브라이트코브 오픈 소스 라이브러리가 포함될 수 있습니다.

    기술: Perl

    https://www.github.com/BrightcoveOS/Perl-Commons

  2. 메트릭 흑연 피클

    Coda Hale의 확장

    기술: 흑연, 자바

    https://www.github.com/BrightcoveOS/metrics-graphite-pickle

  3. LG 샘플 앱

    이것은 Brightcove 계정에서 여러 재생 목록을 표시하는 LG 비디오 응용 프로그램입니다. 이 응용 프로그램은 최소한의 구성이 필요하도록 설계되었으며 고객이 원하는 기능에 따라 UI를 변경하거나 기능을 추가 / 제거하도록 사용자 정의 할 수 있습니다.

    기술: LG, TV

    https://www.github.com/BrightcoveOS/LG-Sample-App

  4. Roku 샘플 앱

    이 템플릿은 Roku에 게시 된 Brightcove 콘텐츠를 얻을 수있는 시작점을 제공합니다. 이 애플리케이션으로 게시를 시작하려면 다음이 필요합니다.

    기술: Roku, TV

    https://www.github.com/BrightcoveOS/Roku-Sample-App

  5. 자바 커먼즈

    이 프로젝트는 다른 Java 프로젝트에서 사용할 수있는 공통 클래스 및 유틸리티를 제공합니다. 여기에는 타사 라이브러리와 브라이트코브 오픈 소스 라이브러리가 포함될 수 있습니다.

    기술: 자바

    https://www.github.com/BrightcoveOS/Java-Commons

  6. 다이아몬드

    Diamond는 시스템 메트릭을 수집하여 Graphite에 게시하는 Python 데몬입니다. CPU, 메모리, 네트워크, I/O, 로드 및 디스크 메트릭을 수집 할 수 있습니다. 또한 거의 모든 소스에서 메트릭을 수집하기위한 사용자 지정 수집기를 구현하기위한 API가 있습니다.

    기술: 흑연, Python

    https://github.com/python-diamond/Diamond
