마이그레이션 문서 Ooyala 고객을 위한 마이그레이션 문서

여기에 나열된 항목은 Brightcove가 비디오 클라우드로 전환하는 동안 Ooyala 고객을 돕기 위해 구축 한 커넥터 및 어댑터를 다룹니다.

참고: 다음 주제가 적용됩니다. 오직 Video Cloud로 마이그레이션한 Ooyala 고객에게

Brightcove는 Ooyala에서 Video Cloud로 마이그레이션하는 데 도움이되는 추가 도구를 개발하고 있으므로 여기를 정기적으로 확인하십시오.

  • Ooyala 플레이어를 사용하여 비디오 클라우드 콘텐츠 제공
  • Brightcove DNS 서버 추가
  • 백롯 / 비디오 클라우드 CMS 커넥터
  • 백롯 및 비디오 클라우드: 비디오 및 재생 목록 필드
  • Backlot API 참조 (Brightcove 버전)
  • 비디오 클라우드로 마이그레이션: 수집 어댑터
    • 비디오 클라우드 수집 어댑터: OIS 2.5로 수집
    • 비디오 클라우드 수집 어댑터: 매니페스트 파일 형식
    • 비디오 클라우드 수집 어댑터: 입력 파일 형식
    • 비디오 클라우드 수집 어댑터: Aspera를 사용한 섭취
    • 비디오 클라우드 수집 어댑터: SFTP를 사용한 수집
    • 비디오 클라우드 수집 어댑터: MRSS를 사용한 수집
    • 비디오 클라우드 수집 어댑터: MRSS를 사용하여 원격 자산 생성
    • 비디오 클라우드 수집 어댑터: CSV 매니페스트 파일
    • 비디오 클라우드 수집 어댑터: XML 매니페스트 파일
  • 비디오 클라우드로 마이그레이션: IQ API 및 비디오 클라우드 분석 API
  • Video Cloud Platform API에 대한 인증
