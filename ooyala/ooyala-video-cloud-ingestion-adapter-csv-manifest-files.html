Oyala-비디오 클라우드 수집 어댑터: CSV 매니페스트 파일

CSV 매니페스트와 함께 FTP 또는 Aspera를 계속 사용하여 비디오 클라우드에 비디오를 수집 할 수 있습니다. 이 항목에서는 CSV의 구조를 설명합니다.
참고: 이 주제가 적용됩니다. 오직 Video Cloud로 마이그레이션한 Ooyala 고객에게

소개

동영상을 수집 할 때 CSV 매니페스트 파일을 사용하여 추가 메타 데이터 (동영상 제목, 설명, 비행 시간, 라벨 및 사용자 지정 메타 데이터 포함)와 미리보기 이미지 및 자막 파일을 지정할 수 있습니다. FTP 또는 Aspera로 콘텐츠를 수집 할 때 CSV 매니페스트 파일을 사용할 수 있습니다.

한계

참고: 이 항목은 OIS v2.5에 적용됩니다.

메타 데이터 파일의 CSV 형식은 XML 형식보다 적은 기능을 지원합니다. CSV 형식을 사용하면 썸네일 (미리보기 이미지) 및 사용자 지정 메타 데이터를 지정하여 비디오와 연결할 수 있습니다. XML 형식은 이러한 기능은 물론 기존에 정의 된 광고 세트의 이름 연결을 지원합니다.

메타 데이터 처리

비디오를 인제스트할 때 비디오를metadata포함하면 다음 두 가지 방법 중 하나로 Video Cloud 비디오 필드에 매핑됩니다.

  1. 기본적으로 각metadata키/값 쌍은 태그 다음 형식으로: meta::key::value . 예를<ooyala:metadata name="category">cooking</ooyala:metadata>들어, 태그를 생성합니다. meta::category::cooking
  2. 메타데이터를 저장하는custom_fields데 를 사용하려는 경우 브라이트코브 지원에서 계정의 설정을 변경하여 이를 수행할 수 있습니다. 다음 사항에 유의하십시오.
    • 메타 데이터 키에 해당하는 이름으로 사용자 지정 필드를 만들어야합니다.
    • 사용자 지정 필드의 수는 기본적으로 10 개로 제한되어 있으므로 메타 데이터를 수용하기 위해 그 이상이 필요한 경우 지원에서 추가 사용자 지정 필드를 요청해야합니다.
    • 각 메타 데이터 항목에 해당하는 사용자 지정 필드가 계정에 없으면 전체 메타 데이터 작업이 실패합니다.

CSV 매니페스트 열 머리글

다음 필드는 CSV 파일의 첫 번째 행 (열 머리글)이며 실제 데이터 필드는 후속 행 (동영상 당 한 행)에 이어집니다.

참고: 이러한 열은 임의의 순서로 지정할 수 있으며 데이터 행의 데이터가 표제와 정확히 일치하는 한 필요하지 않은 열을 생략 할 수 있습니다.

CSV 매니페스트 파일 만들기

CSV 메타 데이터 파일을 생성 할 때 다음 사항에 유의하십시오.

  • 데이터 필드의 텍스트 값에 쉼표 (,) 가 있으면 필드를 큰따옴표 (“) 로 묶어야 합니다.
  • 데이터 필드의 텍스트 값에 큰 따옴표 (“) 가 있으면 필드를 큰따옴표로 묶고 텍스트 필드의 큰 따옴표를 두 번째 따옴표로 이스케이프해야합니다 (예: “그녀는 말했다, “" 좋은 하루 되세요.“"”)

Excel에서 CSV 메타 데이터 파일을 만드는 경우 이스케이프가 자동으로 처리됩니다.

CSV 메타데이터 파일을 만들려면 다음과 같이 하십시오.

  1. 하나 이상의 동영상을 업로드합니다.
  2. Microsoft Excel과 같은 스프레드 시트 프로그램을 엽니 다.
  3. 첫 번째 행에 열 머리글을 추가합니다.
  4. 각 비디오에 대한 행을 추가하십시오.
  5. 파일을 CSV로 저장합니다. 예를 들어파일메뉴에서다른 이름으로저장을 선택할 수 있습니다. 프롬프트가 표시되면 쉼표로 구분 된 값 파일 유형을 선택하십시오.
  6. 선택한 업로드 방법으로 동영상을 업로드 할 때 CSV 파일을 업로드합니다. FTP또는아스페라를 참조하십시오 .
