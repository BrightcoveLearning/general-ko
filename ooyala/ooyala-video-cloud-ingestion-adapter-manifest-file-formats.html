Ooyala-비디오 클라우드 수집 어댑터: 매니페스트 파일 형식

이 항목에서는 비디오 및 자산 수집에 사용할 수있는 매니페스트 파일 형식을 다룹니다.
참고: 이 주제가 적용됩니다. 오직 Video Cloud로 마이그레이션한 Ooyala 고객에게

소개

비디오를 수집 할 때 매니페스트 파일을 사용하여 추가 메타 데이터 (비디오 제목, 설명, 비행 시간, 레이블 및 사용자 지정 메타 데이터 포함)와 썸네일 및 자막 파일을 지정할 수 있습니다. 매니페스트 파일은 XML 또는 CSV 형식입니다. 매니페스트에 대한 자세한 내용은 다음을 참조하세요.

  • CSV 매니페스트 파일
  • XML 매니페스트 파일

처리가 시작될 때

참고: 이 항목은 OIS v2.5에 적용됩니다.
참고: Video Cloud는 XML 또는 CSV 매니페스트 파일이 수신 될 때까지 비디오 처리를 시작하지 않습니다. 따라서 일괄 업로드 전략을 선택하는 것이 중요합니다. 동영상을 빠르게 사용할 수 있어야 하는 경우 (예: 뉴스 배포) 아주 작은 배치를 만들거나 개별적으로 일괄 처리해야 합니다 (동영상당 하나의 CSV 또는 XML 파일).
