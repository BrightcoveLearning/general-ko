소개

미디어 RSS (MRSS)를 사용하여 원격 자산을 추가하거나 업데이트 할 수 있습니다.

MRSS를 사용하여 원격 자산을 추가 또는 업데이트하는 경우, < ooyala:remotese>태그를 지정하여 자산을 원격으로 식별합니다 ( < 미디어:conten> ) 가 MRSS 피드 항목에 있습니다.

참고: 원격 자산을 추가하거나 업데이트할지 여부를 결정하기 위해 Brightcove는<ooyala:embedcode></ooyala:embedcode>필드 XML 매니페스트원격 자산이 현재 존재하는지(업데이트) 또는 존재하지 않는지(추가) 확인합니다.

원격 자산 업데이트

기존 원격 자산의 경우 MRSS를 통해 다음 필드를 업데이트 할 수 있습니다.

  • 미디어:제목
  • 미디어:설명
  • 지역:자막
  • 주야라:메타데이터
  • dcterm:유효한
  • 미디어:축소판
  • 주야라:레이블

MRSS 피드 항목을 제출할 때 항목에 이러한 필드에 대한 업데이트가 포함되어 있으면 <item> 태그를 추가하면 시스템이 적용 가능한 변경 사항으로 원격 자산을 업데이트합니다.
